# Loan Approval Prediction

## Overview

This project predicts whether a loan application should be approved or rejected using machine learning techniques. The model is trained on applicant financial, demographic, and credit-related information.

## Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Visualization
- Model Training
- Hyperparameter Tuning
- Model Evaluation

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Structure

```
Loan_Approval_Prediction.ipynb
README.md
requirements.txt
```

## Dataset

The dataset is **not included** in this repository.
Further Information about the dataset is given in the file "requirements.txt"

## Results

The trained machine learning model predicts loan approval based on applicant information and compares multiple classification algorithms to identify the best-performing model.
